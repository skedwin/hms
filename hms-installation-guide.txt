INSTALLATION GUIDE


MANUAL INSTALLATION
1. Download and extract hms.zip
2. Create new folder in htdocs named it "hms"
3. Copy all extracted files in "hms" folder
4. Create database named "hms" and import hms.sql
5. Open your browser and type in url "localhost/hms" and ka-boom!
